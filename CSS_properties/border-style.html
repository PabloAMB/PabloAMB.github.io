Border-style CSS property

This property can have from one to four values.

border-style: border-top-style border-right-style border-bottom-style border-left-style;

If the property has four values:

  • The first value will affect border-top.
  • The second value will affect border-right.
  • The third value will affect border-bottom.
  • The fourth value will affect border-left.

If the property has three values:

  • The first value will affect border-top.
  • The second value will affect border-right and border-left.
  • The third value will affect border-bottom.

If the property has two values:

  • The fist value will affect border-top and border-bottom.
  • The second value will affect border-right and border-left.

If the property has one value:

  • The value will affect all four borders.

Values

  • none (default)
  • hidden
  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset

Examples

border-style: dotted none solid dashed;
border-style: ridge solid double;
border-style: double groove;
border-style: outset;

This property can affect the four borders, but it can be used in a particular border chosen among:

  • border-top-style
  • border-right-style
  • border-bottom-style
  • border-left-style
