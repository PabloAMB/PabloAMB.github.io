Margin CSS property

This property can have from one to four values.

margin: margin-top margin-right margin-bottom margin-left;

If the property has four values:

  • The first value will affect margin-top.
  • The second value will affect margin-right.
  • The third value will affect margin-bottom.
  • The fourth value will affect margin-left.

If the property has three values:

  • The first value will affect margin-top.
  • The second value will affect margin-right and margin-left.
  • The third value will affect margin-bottom.

If the property has two values:

  • The fist value will affect margin-top and margin-bottom.
  • The second value will affect margin-right and margin-left.

If the property has one value:

  • The value will affect all four margins.

Values

  • lenght (default is 0)
  • %
  • auto

Examples

margin: 23px 2% 14cm 5px;
margin: 10px 50%;
margin: 3em

This property can affect the four margins, but it can be used in a particular margin chosen among:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-right
