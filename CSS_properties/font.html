Font CSS property

This property is a shorthand property for font-style, font-variant, font-weight, font-size and font-family following this order.

font: font-style font-variant font-weight font-size font-family;

Examples

font: italic small-caps lighter small serif;
font: oblique normal larger sans-serif;
font: 12px cursive;
font: italic 20% monospace;

Note: The obligatory properties of those 5, are font-weight and font-family. Which means that you don't need to write the other properties if you don't want to change their values.
