List-style CSS property

This property is a shorthand property for list-style-type, list-style-position and list-style-image following this order.

list-style: list-style-type list-style-position list-style-image;

Examples

list-style: square inside url("image.jpg");
list-style: armenian outside;
list-style: katakana inside;

Note: Values that are missing will have the default value of the property.
