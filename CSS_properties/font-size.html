Font-size CSS property

font-size: size;

Values

  • medium (default)
  • xx-small
  • x-small
  • small
  • large
  • x-large
  • xx-large
  • smaller
  • larger
  • length
  • %

Examples

font-size: 25%;
font-size: small;
font-size: 15px;
font-size: large;
