Border CSS property

This property is a shorthand property for border-width, border-style and border-color following this order.

border: border-width border-style border-color;

Examples

border: 6px double red;
border: dotted green;
border: 7px ridge;
border: dashed;

Notes: The only obligatory property of those 3, is border-style. Which means that you don't need to write the other properties if you don't want to change their values.

This property affects the four borders, but it can be used in a particular border chosen among:

  • border-top
  • border-right
  • border-bottom
  • border-left
