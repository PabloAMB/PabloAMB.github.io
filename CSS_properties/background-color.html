Background-color CSS property

There are five ways to choose a color, but here are the 3 most used:

  1. Write an available color.

    background-color: color-from-colors-list;

    Examples

    background-color: red;
    background-color: green;
    background-color: aqua;

    See colors list clicking here.


  2. Use RGB color.

    background-color: rgb(red, green, blue);

    Examples

    background-color: rgb(165, 165, 236);
    background-color: rgb(26, 105, 28);
    background-color: rgb(0, 239, 255);

    Go to a RGB color picker clicking here.


  3. Use hexadecimal color.

    background-color: #HEX-code;

    Examples

    background-color: #F6E001;
    background-color: #E257DD;
    background-color: #A81818;

    Go to a hexadecimal color picker clicking here.
