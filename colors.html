Colors

There are six ways to choose a color, but here are the 3 most used:

  1. Write color names.

    color-name;

    There are some color names that are accepted in HTML, you can see a list clicking here.

    Examples

    background-color: red;
    border-color: green;
    border-bottom-color: aqua;


  2. Use RGB colors.

    rgb(red, green, blue);

    Go to a RGB color picker clicking here.

    Examples

    background-color: rgb(165, 165, 236);
    border-color: rgb(26, 105, 28);
    color: rgb(0, 239, 255);


  3. Use hexadecimal color.

    #HEX-code;

    Go to a hexadecimal color picker clicking here.

    Examples

    color: #F6E001;
    border-color: #E257DD;
    background-color: #A81818;
